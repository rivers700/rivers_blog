# 2024 年必装的 VS Code 扩展推荐

VS Code 是目前最流行的代码编辑器之一，丰富的扩展生态是它的一大优势。

## 代码质量类

### 1. ESLint

JavaScript/TypeScript 代码检查工具，帮助你保持代码风格一致。

```json
{
  "eslint.validate": [
    "javascript",
    "typescript",
    "javascriptreact",
    "typescriptreact"
  ]
}
```

### 2. Prettier

代码格式化工具，支持多种语言。

**推荐配置：**
```json
{
  "editor.defaultFormatter": "esbenp.prettier-vscode",
  "editor.formatOnSave": true
}
```

## 效率提升类

### 3. GitHub Copilot

AI 代码助手，能够根据上下文智能补全代码。

### 4. Auto Rename Tag

自动重命名配对的 HTML/XML 标签，前端开发必备。

### 5. Path Intellisense

文件路径自动补全，再也不用手动输入路径了。

## 界面美化类

### 6. One Dark Pro

最受欢迎的暗色主题之一，护眼又好看。

### 7. Material Icon Theme

文件图标主题，让文件树更加直观。

## 语言支持类

### 8. Tailwind CSS IntelliSense

Tailwind CSS 智能提示，包括类名补全和悬停预览。

### 9. ES7+ React/Redux/React-Native snippets

React 开发代码片段，快速生成组件模板。

## 其他推荐

### 10. GitLens

Git 增强工具，查看代码历史、blame 信息等。

## 总结

好的工具能够大幅提升开发效率。根据你的技术栈和工作流程，选择适合自己的扩展。

记住：**工具是为了提升效率，不要花太多时间在配置工具上。**
