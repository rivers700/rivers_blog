# ChatGPT Prompt 工程指南

Prompt 工程是与 AI 有效沟通的关键技能。

## 什么是 Prompt 工程？

Prompt 工程是设计和优化输入提示的过程，以获得 AI 模型更准确、更有用的输出。

## 基本原则

### 1. 明确具体

❌ 不好的 Prompt：
> 帮我写代码

✅ 好的 Prompt：
> 用 Python 写一个函数，接收一个整数列表，返回其中所有偶数的平方和

### 2. 提供上下文

```
我正在开发一个电商网站，使用 React + Node.js 技术栈。
请帮我设计一个购物车组件，需要支持：
- 添加/删除商品
- 修改数量
- 计算总价
```

### 3. 指定输出格式

```
请用以下格式回答：
1. 问题分析
2. 解决方案
3. 代码示例
4. 注意事项
```

## 高级技巧

### Few-shot Learning

提供几个示例，让 AI 理解你想要的模式：

```
将以下句子翻译成更正式的表达：

输入：这个东西太贵了
输出：该商品的定价超出了预期范围

输入：我不太懂这个
输出：
```

### Chain of Thought

引导 AI 逐步思考：

```
请一步一步分析这个问题：
1. 首先，理解问题的核心是什么
2. 然后，列出可能的解决方案
3. 最后，选择最优方案并解释原因
```

### Role Playing

给 AI 设定角色：

```
你是一位有 10 年经验的高级前端工程师。
请从专业角度评审以下代码，指出潜在问题和改进建议。
```

## 常用模板

### 代码审查

```
请审查以下代码：
[代码]

请从以下方面给出建议：
- 代码质量
- 性能优化
- 安全性
- 可维护性
```

### 学习新技术

```
我想学习 [技术名称]，我的背景是 [你的技术背景]。
请给我一个学习路线图，包括：
- 核心概念
- 推荐资源
- 实践项目建议
```

## 总结

好的 Prompt 是与 AI 高效协作的基础。多练习、多尝试，你会越来越擅长与 AI 沟通。
