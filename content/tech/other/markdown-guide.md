# Markdown 写作指南

Markdown 是一种轻量级标记语言，让你可以使用纯文本格式编写文档。

## 基础语法

### 标题

使用 `#` 符号来创建标题：

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

- **粗体**: `**粗体文字**`
- *斜体*: `*斜体文字*`
- ~~删除线~~: `~~删除线~~`

### 列表

无序列表：
- 项目一
- 项目二
- 项目三

有序列表：
1. 第一步
2. 第二步
3. 第三步

### 链接和图片

链接: [GitHub](https://github.com)

图片: `![alt文字](图片URL)`

### 引用

> 这是一段引用文字。
> 可以跨越多行。

### 代码

行内代码：`const x = 1;`

代码块：

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

## 数学公式

使用 LaTeX 语法编写数学公式：

行内: $f(x) = x^2$

块级:

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

## 小结

掌握这些基础语法，你就可以开始使用 Markdown 进行写作了！
